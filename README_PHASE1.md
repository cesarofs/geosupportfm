# GeoSupportFM Phase 1 foundations

This bundle adds import-safe modules for:
- AlphaEarth / Satellite Embedding loaders
- Sentinel-2 loaders
- Landsat loaders
- Spatial support transformation utilities
- Raster aggregation tools

Suggested locations in the repo:
- `src/geosupportfm/foundations/alphaearth.py`
- `src/geosupportfm/foundations/sentinel2.py`
- `src/geosupportfm/foundations/landsat.py`
- `src/geosupportfm/support/spatial_support.py`
- `src/geosupportfm/support/raster_aggregation.py`
- `tests/`
